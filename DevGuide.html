Let's Eat! Developer Guideline

By: Brett Tomita, Hayden Matheus, Long Chu, Pamela Regudo, and Jessica Hilario


What is Let's Eat! and its features?

Let's Eat! is an application that assists users in deciding what their next meal is going to be.
Sometimes it seems like there are way too many food options to decide on.
With Let's Eat!, you are guaranteed to find a restaurant nearby that satisfies your needs.

  • Find a Restaurant
  • Vote for a Restaurant
  • Make a Group
  • Rate and Review a Restaurant
  • Pick a Random Restaurant



How to Install?
  • Download Electron
  • If it is your first time running Electron, click here for a first-timer's guide
  • Once set up, head over to our GitHub to download all the files
  • Run the program using your Terminal or Command Prompt



What are the components?

We are using Electron as our framework and NodeJS for our backend to connect with the Yelp API.
Our local server also contains the details of each account created.



How to contribute?

Our source code is located on GitHub It is organized by folders that
indicate where to find source code for certain aspects of our web application.
Feel free to contribute to our source code by cloning our repository and making changes on your local machine.



What is the support you provide?

We offer technical support for users who may have difficulty navigating our website.
We are always open to suggestions from fellow developers as to how our web
application can be improved or ideas for new features for current and future users to enjoy.

Do I need to connect to the Yelp API in order to search for restaurants? How do I do that?
Yes, the Yelp API contains bountiful information about thousands and thousands of restaurants,
it will be inefficient to not use what is already given. Here is a link to a guide.

Where can I get an API Key for Yelp?
You need to create a Yelp account or sign in, go to Manage App
page from their website, then you will receive an email with your API Key.
